Graph plotting and formatting using pysces.plt.*

The simulation and scanning sections described two built-in functions used to plot specific types of data. Currently, all plotting in Matplotlib using the PyLab interface. This interface to focusses heavily on cross-platform interactive plotting and has been designed to use Matplotlib's TKagg back-end. PySCeS makes the Pylab interface available as pysces.plt.P.*
